What is Data, Dataset, and Database?

  • Data: Individual information that we collect and record (e.g., "Ram" (name), "15" (age), "$1" (price))
  • Dataset: Collection of related data pieces (e.g., student list with names, ages, grades)
  • Database: Organized collection of datasets with rules to keep data clean

Lexicographical Sorting

Lexicographical sorting, also known as dictionary order or alphanumerical order, arranges words or sequences of elements in the same way you would find words in a dictionary.

How It Works (ascending ex:)

The comparison process follows these simple rules:

  1. Compare the first elements of the two sequences
  2. If the first elements are different, the sequence with the "smaller" element comes first
  3. If the first elements are the same, move to the second elements and compare them
  4. Repeat this process until a difference is found
  5. If one sequence runs out of elements while the other still has elements, the shorter sequence is considered "smaller"

Why Text Sorting is Slower Than Integer Sorting

Text sorting is generally slower than integer sorting because:

  • Text comparison requires character-by-character analysis
  • Text strings can have variable lengths
  • Text sorting may need to consider case sensitivity and special characters
  • Integer comparison is a single operation per number

Date Storage in Excel

In Excel, dates are stored as serial numbers representing the number of days since January 1, 1900.

Date (What You See) Serial Number (What Excel Sees)
January 1, 1900 1
January 2, 1900 2
September 16, 2025 45916

This allows Excel to perform chronological sorting by simply comparing the serial numbers.

Excel Sorting Algorithms (just learn names only)

Excel uses efficient sorting algorithms like:

Quick Sort

Efficient for most datasets, works by selecting a 'pivot' element and partitioning the array around it

Merge Sort

Stable algorithm good for large datasets, works by dividing the array into halves, sorting them, and then merging

These algorithms are efficient for both integer and text sorting, with Merge Sort being particularly stable for large data sorting.

DAY 2: Data Operations & Shortcuts

Sorting Data

  • Single-level sorting: Organizing data based on one column
  • Multi-level sorting: Organizing data by multiple columns (e.g., first by department, then by marks)

How to Sort

  • Use the Sort tool in the Data ribbon tab
  • Choose column name and sorting order (A-Z or Z-A)
  • Or search for "Sort" in the search bar

SORT Formula

=SORT(array, [sort_index], [sort_order], [by_col])

  • array: The data table you want to be sorted
  • sort_index: The column index to sort by (1 for first column)
  • sort_order: 1 for ascending, -1 for descending

SORTBY Formula (Multi-level Sorting)

=SORTBY(array, by_array1, [sort_order1], [by_array2], [sort_order2], ...)

  • array: The data table you want to sort
  • by_array1: The first column to sort by (instead of index, you use the actual column)
  • sort_order1: 1 for ascending, -1 for descending for the first sort
  • by_array2: The second column to sort by
  • sort_order2: 1 for ascending, -1 for descending for the second sort

Computer Memory

RAM (Temporary Memory)

Excel uses RAM for all operations

Undo and Redo functions use RAM

Volatile - data lost when Excel closes without saving

ROM (Permanent Storage)

Files are saved to ROM (hard drive/SSD)

Data persists even after computer is turned off

Non-volatile - permanent storage

Essential Keyboard Shortcuts

  • Ctrl + X - Cut (removes original data)
  • Ctrl + C - Copy
  • Ctrl + V - Paste
  • Ctrl + Shift + + - Insert row/column before selected one
  • Ctrl + - - Remove selected cells/rows/columns

Selection Shortcuts

  • Ctrl + Shift + Down Arrow - Select range of cells in column with data (if selected the first column cell in the data)
  • Ctrl + Shift + Right Arrow - Select range of cells in row with data (if selected the first row cell in the data)
  • Ctrl + Shift + Down Arrow + Right Arrow - Select the whole data range if the cell you selected is the first cell in column and first row cell in the row
